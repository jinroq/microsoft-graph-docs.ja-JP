# <a name="get-educationrubric"></a>EducationRubric を取得する

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

[EducationRubric](../resources/educationrubric.md)オブジェクトのプロパティとリレーションシップを取得します。

Assignment (`GET /education/me/assignments/{id}/rubric`) を取得する際に返されるものは、に存在する元のの、元のの不変コピーであることに注意`/education/users/{id}/rubrics`してください。 このコピーは、その特定の割り当てに関連付けられています。

## <a name="permissions"></a>アクセス許可

この API を呼び出すには、次のいずれかのアクセス許可が必要です。アクセス許可の選択方法などの詳細については、「[アクセス許可](/graph/permissions-reference)」を参照してください。

| アクセス許可の種類                        | アクセス許可 (特権の小さいものから大きいものへ) |
|:---------------------------------------|:--------------------------------------------|
| 委任 (職場または学校のアカウント)     | EduAssignments、EduAssignments、EduAssignments、および EduAssignments の読み取り/書き込みの基本 |
| 委任 (個人用 Microsoft アカウント) | サポートされていません。 |
| アプリケーション                            | サポートされていません。 |

## <a name="http-request"></a>HTTP 要求

<!-- { "blockType": "ignored" } -->

```http
GET /education/me/rubrics/{id}
```

## <a name="request-headers"></a>要求ヘッダー

| 名前      |説明|
|:----------|:----------|
| Authorization | ベアラー {トークン} |

## <a name="request-body"></a>要求本文

このメソッドには、要求本文を指定しません。

## <a name="response"></a>応答

成功した場合、このメソッド`200 OK`は応答コードと、応答本文で要求された[educationRubric](../resources/educationrubric.md)オブジェクトを返します。

## <a name="examples"></a>例

### <a name="request"></a>要求

要求の例を次に示します。
<!-- {
  "blockType": "request",
  "name": "get_educationrubric"
}-->

```http
GET https://graph.microsoft.com/beta/education/me/rubrics/{id}
```
