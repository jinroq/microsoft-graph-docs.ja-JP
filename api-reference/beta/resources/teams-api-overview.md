## <a name="teams-and-groups"></a><span data-ttu-id="f3dad-134">チームとグループ</span><span class="sxs-lookup"><span data-stu-id="f3dad-134">Teams and groups</span></span>

<span data-ttu-id="f3dad-135">Microsoft Graph では、Microsoft Teams は [group](../resources/group.md) リソースとして表されます。</span><span class="sxs-lookup"><span data-stu-id="f3dad-135">In Microsoft Graph, Microsoft Teams is represented by a [group](../resources/group.md) resource.</span></span> <span data-ttu-id="f3dad-136">Microsoft Teams と Office 365 の両方のグループは、グループの共同作業のためのさまざまなニーズに対応します。</span><span class="sxs-lookup"><span data-stu-id="f3dad-136">Both Microsoft Teams and Office 365 groups address the various needs of group collaboration.</span></span> <span data-ttu-id="f3dad-137">ほとんどのグループベースの機能 (グループの予定表、ファイル、ノート、写真、プランなど) は Microsoft Teams グループと Office 365 グループに適用されます。</span><span class="sxs-lookup"><span data-stu-id="f3dad-137">Almost all the group-based features apply to Microsoft Teams and Office 365 groups, such as group calendar, files, notes, photo, plans, and so on.</span></span> <span data-ttu-id="f3dad-138">[チーム](team.md) と Office 365 のグループの主な違いは、メンバー間のコミュニケション モードです。</span><span class="sxs-lookup"><span data-stu-id="f3dad-138">The main difference between a [team](team.md) and an Office 365 group is the mode of communication between members.</span></span> <span data-ttu-id="f3dad-139">チーム メンバーのコミュニケーションには、特定のチームのコンテキストで常設チャットが使用されます。</span><span class="sxs-lookup"><span data-stu-id="f3dad-139">Team members communicate by persistent chat in the context of a specific team.</span></span> <span data-ttu-id="f3dad-140">Office 365 グループのメンバーのコミュニケーションには、グループ会話 (Outlook のグループのコンテキストで行われるメールによる会話) が使用されます。</span><span class="sxs-lookup"><span data-stu-id="f3dad-140">Office 365 group members communicate by group conversations, which are email conversations that occur in the context of a group in Outlook.</span></span>
