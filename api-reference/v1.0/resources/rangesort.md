# <a name="rangesort-resource-type"></a>RangeSort リソースの種類

Range オブジェクトの並べ替え操作を管理します。


## <a name="methods"></a>メソッド

| メソッド           | 戻り値の型    |説明|
|:---------------|:--------|:----------|
|[Apply](../api/rangesort-apply.md)|None|並べ替え操作を実行します。|
